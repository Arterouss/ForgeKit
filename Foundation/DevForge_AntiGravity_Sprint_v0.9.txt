Tool Engine Foundation

Continue the existing DevForge project.

IMPORTANT - Do NOT recreate the application. - Do NOT redesign previous
UI. - Keep existing architecture. - Extend only.

================================================== MISSION
==================================================

Build the Tool Engine that every future utility will use.

This is the foundation for all tools.

Every tool must follow the same architecture, layout and interaction
patterns.

Think of this as an internal framework.

================================================== GOALS
==================================================

Create a reusable Tool Engine.

Every future utility should be created by plugging into this engine.

================================================== TOOL PAGE LAYOUT
