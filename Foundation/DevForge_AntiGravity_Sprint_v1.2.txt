================================================== FEATURES
==================================================

-   Decode JWT automatically
-   Validate token structure
-   Display Header
-   Display Payload
-   Display Signature
-   Pretty JSON formatting
-   Copy Header
-   Copy Payload
-   Copy Full JSON
-   Paste Token
-   Clear Token

================================================== VALIDATION
==================================================

Detect: - Invalid format - Missing segments - Invalid Base64 - Empty
token

Show friendly messages.

================================================== SECURITY
==================================================

Everything runs locally.

Never send tokens to external APIs.

Display a warning that decoding does not verify authenticity.
