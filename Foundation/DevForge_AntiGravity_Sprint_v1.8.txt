================================================== MISSION
==================================================

Build a visual Docker Compose Builder for developers.

The tool should help users generate production-ready docker-compose.yml
files without writing YAML manually.

================================================== CORE FEATURES
==================================================

-   Visual service builder
-   Live YAML generation
-   YAML validation
-   Copy
-   Download docker-compose.yml
-   Import existing docker-compose.yml
-   Clear project
-   Load templates

================================================== SERVICE CONFIGURATION
==================================================

Support configuring:

-   Image
-   Container Name
-   Build Context
-   Ports
-   Environment Variables
-   Volumes
-   Networks
-   Restart Policy
-   Depends On
-   Healthcheck
-   Command

================================================== SUPPORTED SERVICES

Provide presets for:

-   PostgreSQL

-   MariaDB
-   Redis

-   Traefik
-   Node.js
-   Next.js
-   Laravel
-   WordPress

================================================== LIVE PREVIEW
==================================================

Generate docker-compose.yml in real time.

Use Monaco Editor.
